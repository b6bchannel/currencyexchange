# Currency — 极简新拟态多向汇率看板

一个完全运行在浏览器端、无广告的私人专属汇率转换 PWA (Progressive Web App) 应用。专为高频跨国出行与个人多币种资产看板量身定制。
---

## ✨ 核心设计与交互特性

### 🎨 新拟态 (Neumorphism) 视觉美学
* **材质层级**：彻底摒弃传统生硬的边框线条，完全依靠纯正的光影高光与软阴影（Soft Shadows）构建浮雕般沉浸式的视觉呼吸感。
* **高对比度输入区**：数字输入框单独调配了深色冷灰材质（`#D5DFEB`），与卡片背景完美区分，层级清晰。

### 📱 微信级原生手势轨道
* **多向联动计算**：摒弃传统“先选币种、再输金额”的死板逻辑。点击任意卡片即可直接唤起手机数字键盘，输入的同时其他所有币种卡片**毫秒级动态同步更新**。
* **双层手势解耦**：
  * **右滑卡片**：整体平移并优雅浮现**红色删除键 (✕)**。
  * **左滑卡片**：整体平移并平滑露出**绿色添加键 (+)**。
* **无误触设计**：彻底删除了卡片上常驻的易误触按键，极大拓宽了点击区域。
* **防干扰锁死**：硬件级注入 `touch-action: pan-y`，完美规避 iOS 系统的横向回弹与侧滑干预，完美顺滑。

### 🌐 环球时间轴
* 每个货币卡片下方轻量级集成了**当地的精确系统时间**，自动解析 `AM/PM` 格式，并附带标准的 `GMT` 时区偏移量（如：中国东八区、柏林时间、悉尼时间、纽约时间等）。

### 🛠️ 极致的企业级稳定保底
* **双节点无缝容灾**：底层聚合了 `jsDelivr CDN` 主节点与 `Cloudflare` 备用节点。在中国大陆等复杂网络环境下，一旦主节点解析超时，将在底层无感自动静默切换。
* **本地持久化存储 (LocalStorage)**：自动记忆你的**币种留存、排序状态以及上一次退出的输入金额**。
* **零延迟离线运行**：每次拉取数据后会自动在手机本地备份完整汇率表。即使在飞机或无网状态下，也能秒开 App 并使用离线数据完美计算。

## 📲 如何在 iPhone 上快速安装？

1. 在手机上使用自带的 **Safari 浏览器** 打开你的 GitHub Pages 专属链接。
2. （可选）首次打开若未正常抓取图标，请**强制刷新一次网页**以清理缓存。
3. 点击 Safari 底部工具栏正中间的 **“分享” 按钮**（带有向上箭头的方块）。
4. 在弹出的系统菜单中向下滚动，找到并点击 **“添加到主屏幕” (Add to Home Screen)**。
5. 在右上角点击“添加”，回到手机桌面即可开启完美隐藏浏览器地址栏的**纯全屏原生 App 体验**。
