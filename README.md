# Currency

一个基于新拟态 UI 的简易多向汇率转换 Web App。专为 iOS Safari 设计，支持作为 PWA（渐进式网页应用）添加到主屏幕全屏使用，无广告，不依赖付费开发者账号。

---

## 功能特性

* **新拟态 UI**：完全依靠高光和软阴影构建空间感，数字输入框使用加深色调（#D5DFEB）以增强视觉层级。
* **多向联动**：点击任意币种输入框即可直接唤起手机数字键盘，输入金额时，其他所有卡片的数据毫秒级同步计算。
* **微信级手势操作**：卡片无常驻易误触按钮。左滑卡片平移露出**绿色添加键 (+)**，右滑卡片平移露出**红色删除键 (✕)**。
* **排序与刷新**：右上角配备独立按钮，支持手动调整卡片前后顺序，以及手动点击 ↻ 强制拉取最新汇率。
* **防干扰处理**：注入 `touch-action: pan-y` 属性，彻底锁死 iOS 浏览器的横向回弹与系统侧滑干预。
* **世界时间集成**：各币种卡片内嵌当地精确系统时间，自动解析 AM/PM 格式并附带 GMT 时区偏移量。
* **双节点容灾与离线模式**：
  * 聚合 jsDelivr CDN 主节点与 Cloudflare 备用节点，主节点请求超时自动无感切换。
  * 首次加载后将汇率表、卡片顺序及最后一次输入的数值存入本地 LocalStorage。无网状态下依然可以秒开 App 并使用离线数据完美计算。

## 已适配币种

支持在面板中自由组合、排序以下币种：
* CNY (人民币)、USD (美元)、EUR (欧元)、JPY (日元)、GBP (英镑)、HKD (港币)、TWD (新台币)、SGD (新加坡元)、AUD (澳元)、CAD (加币)、VND (越南盾)、THB (泰铢)、KRW (韩元)、PLN (波兰币)、CHF (瑞士法郎)、NZD (新西兰元)

## 安装到桌面步骤

1. 在 iPhone 上使用 **Safari 浏览器** 打开你的 GitHub Pages 链接。
2. （若图标未正常抓取，请强制刷新一次页面）。
3. 点击 Safari 底部工具栏正中间的 **“分享” 按钮**（带有向上箭头的方块）。
4. 在菜单中向下滚动，点击 **“添加到主屏幕” (Add to Home Screen)**。
5. 点击右上角“添加”，即可在桌面生成高清图标并隐藏浏览器地址栏全屏使用。
